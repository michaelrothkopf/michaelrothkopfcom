Editor

Thank you for using my quiz editor! To start, simply input the name of the quiz into the "Quiz Name" box or click "Import Quiz" at the bottom to work on an exiting quiz. Then, use the bar on the left to select the question you want to work on. You can create new questions by pressing "New Question" at the bottom of the left-hand bar and delete questions using the delete button in each question's editor. If you want to share your quiz, simply click "Copy Quiz Data" and it will be copied to the clipboard, which you can then post to a document to be used by others.
